# Apollo’s Learning Corner — Mini Brand Style

## Brand Name
Apollo’s Learning Corner

## Look & Feel
- **Backgrounds:** clean white / very light gray
- **Primary Colors (bright):**
  - Sky Blue (#4DA3FF)
  - Sunshine Yellow (#FFD84D)
  - Coral Red (#FF6B6B)
  - Leaf Green (#4CD964)
  - Purple (#8E5DFF)
- **Accent:** Charcoal text (#2B2B2B)

## Typography (suggested)
- **Headings:** Fredoka / Baloo / Nunito (rounded, kid‑friendly)
- **Body:** Nunito / Quicksand / Arial
- **Sizing:** H1 34–42, H2 22–28, Body 12–14

## Visual Motifs
- Simple bold shapes (circle, square, triangle)
- Thick outlines (2–4px) with bright fills
- Smiley faces on shapes for friendliness

## Voice/Tone
- Friendly, simple, encouraging
- Short phrases, direct instructions

## Default CTA
“Free download → Link in bio”
