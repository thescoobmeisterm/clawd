# Apollo's Learning Corner - Video Production Checklist

## Pre-Production ✓

### Script Review
- [ ] Age-appropriate vocabulary (4-8 year olds)
- [ ] Clear learning objective stated
- [ ] 1 concept per video (avoid overload)
- [ ] Engaging hook in first 3 seconds
- [ ] Repetition for key concepts (3x minimum)
- [ ] Clear conclusion/call-to-action

### Visual Planning
- [ ] Background assets ready
- [ ] Character animations queued
- [ ]Props/icons for concepts
- [ ] Smooth transitions planned
- [ ] Colors: Bright, kid-friendly, high contrast

### Audio Setup
- [ ] ElevenLabs TTS voice selected
- [ ] Script synced to audio length (3-5 min ideal)
- [ ] Background music level set (-20dB below voice)
- [ ] Sound effects ready for emphasis

## Production ✓

### Recording/Creation
- [ ] Video rendered at 1080p minimum
- [ ] 30fps for YouTube, 60fps for Shorts
- [ ] Audio normalized to -14 LUFS
- [ ] Captions burned in or as separate file

### Quality Check
- [ ] No dead air or awkward pauses
- [ ] Audio clear, no clipping
- [ ] Visuals match narration timing
- [ ] Branding (Apollo logo) included

## Post-Production ✓

### YouTube Upload
- [ ] Optimized title (keyword + hook)
- [ ] Description with timestamps & links
- [ ] Tags: kids learning, preschool, education, [concept]
- [ ] Thumbnail: bright, large text, character visible
- [ ] Playlist: add to appropriate series

### Shorts Format (under 60 sec)
- [ ] Vertical aspect ratio (9:16)
- [ ] Hook in first 2 seconds
- [ ] Fast-paced visuals
- [ ] End card linking to full video
