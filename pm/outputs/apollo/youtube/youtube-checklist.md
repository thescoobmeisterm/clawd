# YouTube Channel Quick‑Win Checklist (Kids Learning)

## Channel Basics
| Item | Why | Done |
|---|---|---|
| Clear banner text: “Ages 3–7 Learning Videos” | Instant clarity | ☐ |
| Channel description w/ 3 keywords | Helps discovery | ☐ |
| Add channel trailer (30–45s) | Explains value fast | ☐ |
| Upload channel icon (simple, bold) | Recognition | ☐ |
| Add links (website, printable) | Conversion | ☐ |

## Content Setup
| Item | Why | Done |
|---|---|---|
| Default video template (title style) | Consistency | ☐ |
| Create 3 playlists (Colors, Shapes, Numbers) | Easy browsing | ☐ |
| Pin comment with printable link | Convert viewers | ☐ |
| End screen → “Next video” | Retention | ☐ |
| Cards at 30s + 60s | Keep viewers | ☐ |

## Shorts Growth
| Item | Why | Done |
|---|---|---|
| 3 Shorts/week minimum | Feed momentum | ☐ |
| 1 hook template repeated | Fast production | ☐ |
| Add subtitles (big, 2–4 words) | Retention | ☐ |
| CTA: “Free printable in bio” | Lead gen | ☐ |
| Reuse long‑form clips | Efficiency | ☐ |

## SEO Quick Wins
| Item | Why | Done |
|---|---|---|
| Title format: “Color + Shape Hunt | Ages 3–7” | Search clarity | ☐ |
| First 2 lines of description include keywords | Ranking | ☐ |
| 3–5 tags (colors, shapes, preschool, toddlers) | Discoverability | ☐ |
| Add chapters (if 1+ min) | UX | ☐ |

## Branding & Trust
| Item | Why | Done |
|---|---|---|
| Same font + colors across thumbs | Brand recall | ☐ |
| “Made for parents” line in description | Trust | ☐ |
| Comments pinned with “What did your kid find?” | Engagement | ☐ |

## Weekly Routine (15 min)
| Task | Why |
|---|---|
| Review top 3 videos CTR + retention | Double down | 
| Add 1 new playlist item | Organization | 
| Reply to 5 comments | Trust |
