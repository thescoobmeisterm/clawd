# YouTube Channel Checklist (Apollo)

## Brand + Setup
- [ ] Channel name + handle locked
- [ ] About section: 1‑sentence mission + audience
- [ ] Link to main site / lead magnet
- [ ] Channel icon (clear at small size)
- [ ] Banner (safe zone tested)
- [ ] Default channel trailer (30–60s)

## Video Standards
- [ ] Consistent intro style (≤3s)
- [ ] Consistent outro CTA (subscribe + lead magnet)
- [ ] Title format template saved
- [ ] Thumbnail style guide (fonts/colors)
- [ ] Upload preset (tags + description)

## SEO + Metadata
- [ ] 5–10 keyword phrases documented
- [ ] Title includes main keyword + hook
- [ ] First 2 lines of description include CTA + link
- [ ] Tags include keyword variations
- [ ] Hashtags included (2–3)

## Content Cadence
- [ ] 4‑week content calendar drafted
- [ ] Batch record schedule set
- [ ] Weekly Shorts plan (2–3)

## Publishing Workflow
- [ ] Upload checklist saved
- [ ] End screens template configured
- [ ] Cards template configured
- [ ] Pinned comment template created

## Growth Hooks
- [ ] 1 lead magnet offer live
- [ ] Community tab usage plan
- [ ] 3 collaboration targets listed

## Metrics to Review Weekly
- [ ] CTR (thumbnails/titles)
- [ ] Avg view duration
- [ ] Subs gained per video
- [ ] Top traffic sources
- [ ] Best‑performing keywords
