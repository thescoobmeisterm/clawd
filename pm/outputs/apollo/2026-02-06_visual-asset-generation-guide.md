## Style Consistency Rules

| Element | Specification |
|---------|---------------|
| Resolution | 1920x1080 minimum (for YouTube) |
| Aspect Ratio | 16:9 for full video, 9:16 for Shorts |
| Background | White or transparent for overlays |
| Lighting | Bright, even, studio-style |
| Style | 3D render, Pixar-like, clean edges |
| Colors | Primary + secondary, high saturation |
| File Format | PNG with transparency |

## Production Workflow

```
┌─────────────────────────────────────────────────────────────┐
│  1. Generate AI Images (Leonardo.ai or preferred tool)     │
│     - Use prompts above, adjust for style consistency     │
│     - Generate 2-3 variations per number                   │
│     - Select best, upscale to 1080p                        │
├─────────────────────────────────────────────────────────────┤
│  2. Export PNG Files                                       │
│     - Ensure transparent backgrounds where needed          │
│     - Name consistently: 1_block.png, 2_stars.png, etc    │
├─────────────────────────────────────────────────────────────┤
│  3. Import to Video Editor (CapCut / Premiere)             │
│     - Layer visuals with TTS audio                         │
│     - Add subtle animations (bounce, float)                │
│     - Match timing to script markers                       │
├─────────────────────────────────────────────────────────────┤
│  4. Add Background Music                                  │
│     - Upbeat but not overwhelming                          │
│     - -20dB below voice level                             │
│     - Space-themed fits Apollo brand                       │
├─────────────────────────────────────────────────────────────┤
│  5. Export and Upload                                     │
│     - YouTube: 16:9, 1080p                                │
│     - Shorts: 9:16 vertical cut                           │
│     - Thumbnail: Bright, large number, Apollo visible     │
└─────────────────────────────────────────────────────────────┘
```
