# AI Image & Design Work — Prompt Library v1

## Prompt Formula (Base)
**Subject + Style + Composition + Lighting + Mood + Palette + Output Specs**

**Template:**
> {subject}, {style}, {composition}, {lighting}, {mood}, {palette}, {camera/lens}, {render quality}, {background}, {artifacts to avoid}

## Style Buckets (Use as tags)
- Minimalist vector
- Flat kids illustration
- Retro 80s/90s
- Geometric/abstract
- Hand‑drawn sketch
- Watercolor storybook
- Bold streetwear
- 3D clay/toy

## Output Specs (for print + merch)
- 300 DPI
- Transparent background (when possible)
- Safe area margin (10%)
- 2–3 color variants per prompt

## Negative Prompt Ideas (quality control)
- “blurry, low‑resolution, extra limbs, deformed, watermark, text, logo, signature, artifacts, noise.”

## Usage Tips
- Start broad → narrow with 1–2 modifiers.
- Keep palette consistent per collection.
- Always generate 2–3 variations.
